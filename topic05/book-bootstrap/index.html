Objectives

The page links to a CSS library called 'Bootstrap':

  • http://getbootstrap.com

This is one of the most popular CSS frameworks. It is designed to be hand coded:

  • http://getbootstrap.com/getting-started/

so most of the classes used are reasonable meaningful.

Spend a few minutes perusing the documentation - and see if you can identify any of the class we are using in the above example.

Forms

On homeprofile, change form as follows:

          <div class="ui column form segment">
            <form action="homeprofile/changetext" method="post">
              <textarea name="profiletext"> </textarea>
              <input type="submit" name="submit" value="Change" />
            </form>
          </div>

change button:

              <input class="ui blue button" type="submit" name="submit" value="Change" />


              <textarea class="ui field" name="profiletext"> </textarea>

Put in a header:

          <div class="ui column form segment">
            <h3 class="ui block header"> Enter Status</h3>
            <form action="homeprofile/changetext" method="post">
              <textarea class="ui field" name="profiletext"> </textarea>
              <input class="ui blue button" type="submit" name="submit" value="Change" />
            </form>
          </div>

change header color:

            <h3 class="ui teal block header"> Enter Status</h3>

Invert it:

            <h3 class="ui inverted teal block header"> Enter Status</h3>
